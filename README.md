# Todo App with etcd backend 

## Build & Run

**Build**
``` bash
make build
```

**Run**

1. Create a .env file and add Github Client ID and Secret like follows
```
GITHUB_CLIENT_ID=<YOUR_GITHUB_CLIENT_ID>
GITHUB_CLIENT_SECRET=<YOUR_GITHUB_CLIENT_SECRET>
```

2. Run the service
``` bash 
make run
```
